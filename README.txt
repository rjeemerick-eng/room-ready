========================================
  ROCK KIDS CLASSROOM BOARD
========================================

FIRST TIME SETUP (do this once):
  Mac:     Double-click "SETUP (Run Once).command"
  Windows: Double-click "SETUP (Run Once).bat"

  This installs the app. Takes about 1–2 minutes.
  Requires an internet connection for first-time setup only.

EVERY SUNDAY:
  Mac:     Double-click "START Rock Kids.command"
  Windows: Double-click "START Rock Kids.bat"

  The board window opens automatically.
  A pop-up will show the link for volunteers.

NDI SCREEN CAPTURE SETUP:
  1. Download NDI Tools (free) from: https://ndi.video/tools/
  2. Open "NDI Screen Capture" from the NDI Tools package
  3. Select the "Rock Kids – Classroom Board" window
  4. In your switcher (vMix, OBS, Wirecast), add an NDI source
     and select the Rock Kids capture

VOLUNTEER PHONES:
  When the app starts, a pop-up shows the volunteer link.
  You can also find it in the tray icon menu (bottom right on Windows,
  top right on Mac) → "Show Volunteer Link"

  Volunteers open that link on their phone browser — no app needed.
  They flip the toggles and tap "Update Board".
  The display refreshes automatically within 5 seconds.

  NOTE: For security, the Settings page (branding, colors, NDI,
  Planning Center) only works on the board computer itself.
  Volunteer phones can only change room statuses.
